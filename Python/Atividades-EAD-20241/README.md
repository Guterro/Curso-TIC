# Atividades EAD de Algoritmos e Programação (20241)
Os projetos anexados scontém as atividades desenvolvidas ao longo do meu primeiro semestre na disciplina de Algoritmos e Programação (2024.1). Os exercícios abrangem desde operações matemáticas básicas até estruturas de dados mais complexas como matrizes e registros (POO).

---

## Conceitos de Programação Aplicados

O projeto demonstra o domínio dos seguintes fundamentos de Python:

* **Operações Básicas e Cálculos Dinâmicos:** Manipulação de variáveis e operadores aritméticos para soluções do cotidiano, como calculadoras, cálculos de consumo de combustível e rendimentos financeiros.
* **Sub-rotinas (Funções):** Modularização de código para aumentar o reaproveitamento e a organização, incluindo lógica de funções matemáticas (fatorial) e comparadores de valores.
* **Vetores (Arrays):** Armazenamento e manipulação de listas unidimensionais, aplicando filtros de dados, buscas de menor/maior valor e substituições condicionais em conjuntos de dados.
* **Registros (Orientação a Objetos):** Introdução à POO com o uso de classes e objetos para estruturar dados complexos, como sistemas de cadastro de alunos e o desenvolvimento de uma agenda telefônica interativa com funções de busca e remoção.
* **Matrizes:** Trabalho com estruturas bidimensionais para organização de dados em tabelas, incluindo preenchimento aleatório, cálculos de soma total em matrizes e operações de multiplicação por escalar.

---

## Tecnologias Utilizadas

* Biblioteca NumPy (para manipulação de arrays e registros)
* Módulo Random (para simulações de dados)
