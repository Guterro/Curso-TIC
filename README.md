# Projetos Acadêmicos - UFSC

Este repositório centraliza os projetos, exercícios e trabalhos desenvolvidos durante minha graduação em Tecnologia da Informação e Comunicação (TIC) na UFSC - Campus Araranguá.

## Organização do Repositório

Os projetos estão organizados por disciplinas e tecnologias:

### [POO-Java](./POO-Java)

* **Descrição:** Aplicação de conceitos de Herança, Polimorfismo, Encapsulamento e Abstração.
* **Tecnologias:** Java.

### [Seminarios-Python](./Seminarios-Python)
* **Descrição:** Scripts e algoritmos desenvolvidos para apresentações e seminários de lógica de programação.
* **Tecnologias:** Python 3.
